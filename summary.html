ABOUT
Sound in this project is measured as anything the microphone on your device picks up.It can be intentional or unintentional, and sound varies depending on device and microphone.
Decibels, shortened as dB,is a unit used to measure the intensity or volumne of sound on a scale. Something 10 dB louder is actually 10 times more intense.
Hertz, shorted as Hz,is a unit used to measure the frequency of sound waves. This is the number of cycles per second a sound wave completes.
A hexcode is a six letter or number code that refers to a color. The first two letters/numbers refer to red, the next two refer to green, and the last two refer to blue. The color values are defined in values between 00 and FF, representing 0 to 255. A-F are for 10-15. Each value is translated into the digit it represents. The first of of the two digit code for each color is multiplied by 16^0. The second of the two digit code is multiplied by 16^1. These are then added together to get the value for the color.
